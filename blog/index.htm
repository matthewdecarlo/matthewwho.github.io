Coding Loops

Ruby v. JavaScript

For-loop

Ruby has some cool tricks where you can enumerate with built in methods. In JavaScript there is no such luck! The following cheat sheet should help bridge the gap for those who learned to code with Ruby and are picking up JavaScript. Good luck, you're going to need it!

With the for-loop in Ruby, the use of an array allows for the loop iterate over the array. Once for each element.

Ruby: for-loop
$ for a_number in ( [1, 2, 3, 4, 5] ) do
$ puts "The current number in the array is: #{a_number}"
$ end

=> The current number in the array is: 1
=> The current number in the array is: 2
=> The current number in the array is: 3
=> The current number in the array is: 4
=> The current number in the array is: 5

The for-loop in JavaScript requires an integer variable be initialized, followed by a comparison, and then a modification to the integer variable. In this example, the modification increases the number by one at each pass of the for-loop.

JavaScript: for-loop
> for (var aNumber = 1; aNumber <= [1,2,3,4,5].length; aNumber +=1) {
> console.log("The current number in the array is: " + aNumber)
> };

   The current number in the array is: 1
   The current number in the array is: 2
   The current number in the array is: 3
   The current number in the array is: 4
   The current number in the array is: 5

Each/ forEach loop

Enumeration is a beautiful thing. If you want to get each value or of an array the each method in Ruby or the forEach function in JavaScript is your guy!

With each of the values in the array, Ruby passes one value from the array into the code block. As Ruby does this it creates a new local variable within the instance of the block of code. The each loop continues to iterate until it has passed each value from the array. (Note: hashes in Ruby also have an each enumerator method.)

Ruby: each-loop
$ ['one', 'two', 'three', 'four', 'five'].each do |a_word|
$ puts " The current word in the array is: #{a_word}"
$ end

=> The current word in the array is: one
=> The current word in the array is: two
=> The current word in the array is: three
=> The current word in the array is: four
=> The current word in the array is: five

The JavaScript forEach function only works on arrays. It calls a function for each value in an array and runs the function's code block. In this example the code block prints the value once per each iteration.

JavaScript: forEach-loop
> ['one', 'two', 'three', 'four', 'five'].forEach(function(aWord){
> console.log("The current word in the array is: " + aWord)
> });

   The current word in the array is: one
   The current word in the array is: two
   The current word in the array is: three
   The current word in the array is: four
   The current word in the array is: five
